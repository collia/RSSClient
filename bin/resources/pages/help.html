Rss Reader is a program to read RSS and Atom news feeds.
RssReader collect news in the background at user configurable intervals. You can click the server icon to see a short description of the news and click or open the original news web page in default browser window. RSS (Really Simple Syndication) is a format for syndicating news. Many news and web log sites support RSS by adding these pictograms as XML or RSS to their web pages. RssReader support RSS versions 0.9x, 1.0 and 2.0 and Atom 0.1, 0.2 and 0.3. Running under Windows 98/NT/Me/2000/XP/2003/Vista/2008 and Windows 7.
